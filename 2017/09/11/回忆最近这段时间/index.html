回忆最近这段时间

发表于 2017-09-11 | 分类于 越早醒悟越好 | | 阅读次数

最近的几个月里都在忙里奔波，少记录很多事情。还好该记得的，我都记在心里了。此时此刻，我觉得那段美好的时刻，应该被记录下来，供日后回忆好久好久。

过去的很长一段时间里，我一直认为自己不会选择下厂子干活，至少应聘工作的时候，不会去考虑类似的公司，在我自己的设想里面，我是应该敲着键盘的IT男，并没日没夜的与bug作斗争。不单单是我的设想，还是我在梦中拉出来的理想。

如果时间以月节点，我想我所有的努力可能不值一提，但至少我会为这个选择认真过。在2017年的5月的这段日子里，我找人生中第一份开发工作——前端工程师。真正开始尝试做起企业开发工作。并已经开始建立起自己的github、blog记录着历程。虽然不多，但里面含着真挚的情感。

早在4个月前，我依旧还是那个迷茫的我，不知从何开始选择，毕竟一个刚毕业的大专生可供的选择不可能太多。心里总会想着，想做的未能能够做到，毕竟跨了一个专业，基础又未必扎实，而且还缺少项目经验，而目前的IT行情，对新人的要求也越来越高。后面的可想而知，面临的只有妥协，找一份接线工作，然后慢慢开始做起。但我知道这并不是我想要的，依旧每天都会去寻求改变，但又不知道路径，只好每天尝试的查找各种各样的资料，自己慢慢的找准方向开始做起。。那时在实训室的日子，便随我的唯有自己，但内心还是不够坚定。

对于一个初学者来说，待续。。。（上文主要体现逐渐想社会现状作出妥协，而又无法适从的心里）

多上可以从上文看出自己当时处在一个努力未得到理想效果的心里偏差，是的，目前回过头来看，其实慢慢可以看出心里与其纠结这件事情，不如花多点时间去做好自己想做的事情。

从某种… （过去的，还是让它随风飘逝吧，当下也有选择，也需要坚持）
