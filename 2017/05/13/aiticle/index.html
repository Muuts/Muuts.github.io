独立博客

发表于 2017-05-13 | 分类于 生活不止苟且，还要有诗和远方 | | 阅读次数

​ 在很早一开始，创建本博客的初衷，为了记录与学习，想把些亲身踩过的坑 ，对技能的见解，以及在这一路上的总结，给记录和分享在博客上。同时希望自己的想法能有一个寄托。之所以选择，大概是因为看的多了(blog)，就会有种想选择的想法，理由虽有些牵强，不过总是实践出来了(这里要3q辉爷的赶脚)~。

​ 我的第一篇博文，本就打算记录下这周上班所遇到问题(bug)。仔细想想一开始就推些不知所云的见解~~~,怎么想都不合适。所以还是选择抒抒情（chui shui），比如自己在选择职业规划上，是怎么做出思考的。以及在作为一个职场新人，做过哪些很纯的事情呀~ 但其中一点很重要，这篇文章，我会按我几个阶段来写，并不是一次性全写完，大概一周会有一个小篇幅。毕竟想要总结（chui shui）的东西太多。时间上也不允许等，我就不一一举例了。并且说明这并不是一篇鸡汤文，只想做为生平的一次小记录。

​ 感慨下：作为一个刚入职不久的新人， 工作开始还是挺有压力的，起码头三天，工作时间内，满脑子都是需求的问题。其中，很重的一点，不知道新人入公司会不会有一种想法，一开始接任务的时候，着实在真没把握，基本有个短时间都在担心，头脑里面如果实现不了需求该如何处理，这个短时间内，估计都有的来想了~（一起期待下周的小篇幅）
