Sunshine

发表于 2017-05-30 | 分类于 生活不止苟且，还要有诗和远方 | | 阅读次数

今早非常有幸的加到了@花仲马的微信，真的非常有幸。而且还使坏的问了一句，花花姐什么时候嫁给芬达大神。哈哈，估计花花姐现在还在加班。没空回复我。看过phodal博客的朋友都知道，这是一段非常有浪漫色彩的爱情短跑，尤其是那句（待我代码编成，娶你为妻可好@花仲马），简直每逢看到phodal的文章时，总能想到这句。这引发了我的思考，等我代码写好，上哪找我的花仲马呢?

但是我觉得任何时候都需要给自己信心，毕竟这三年走过路，都是通过自己努力去争取的。路边的每一段风景，虽说不是格外绚丽灿烂，少有人陪。我想每条路都有属于自己的归属。路的尽头，也最终将迎来毕业的到来。无论之前故事如何，但这次我任将选择重头开始。白驹过隙，忽然而已。

现在经历还太少，总结来说，生活就像代码，总能遇到始料不及的bug。

这周工作的总结，我收集了各种了移动端的框架比较，还有把之前一些代码上传带github，模仿ionic京东购物车，以及尝试去还原公司的首页滚动效果。但是做的并不好，没有研究透，而这假期一直处于游玩的状态，还是自身的学习压力不够太，导致对自己的要求松懈了。以及设想的都没有做，比较懊恼。 在这里提主要是希望自己能改变这种状态，毕竟自身的定位是明确的，把之前那种学习劲持续持下去。

而端午节假期去了白云山游玩，印象深刻的不是他的美景，而是山里面的蚊子和索道缆车，尤其是蚊子，咬一口留下的印记三天不退，每观望一次就回忆一下，久久不能忘怀。在持续性的走动一般是不会被蚊子咬的，因为在看别人蹦极的时候，逗留的太久，给了蚊子可乘之机，被咬了个措不及防，实在可恨。还有就是第一次坐索道缆车，这次体验又刷新了对事物感知，真是个难得的体会，毕竟工作后这种机会真的不多了，能狼一次算一次，记录下了也算是对远方的一种寄托吧~
